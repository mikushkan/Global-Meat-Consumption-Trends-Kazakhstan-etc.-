Global Meat Consumption Trends Analysis

# About the Project
This project was developed during my studies in Germany as part of the course World in Data. The analysis focuses on global meat consumption trends and explores how dietary patterns differ across countries and change over time.

# Using historical food consumption data, the project investigates:
- total meat consumption per capita,
- differences between countries,
- changes in meat consumption structure,
- correlations between meat types,
- long-term consumption trends.
- The analysis pays particular attention to Kazakhstan and compares it with countries such as Russia, the United States, China, Brazil, Germany, and Japan.

# Dataset
Source: Our World in Data / FAO Food Consumption Statistics
Dataset includes:
- beef and buffalo meat consumption
- poultry consumption
- pork consumption
- sheep and goat meat consumption
- other meat categories
Measured in kilograms per capita per year.

# Project Goal
The purpose of this project is to demonstrate practical data analysis and visualization skills in R, including:
- data cleaning,
- transformation,
- exploratory data analysis,
- comparative analytics,
- visualization of historical trends.
